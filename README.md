# Visit to Berlin

## Overview
"Visit to Berlin" is an HTML/CSS web app designed to **showcase the city of Berlin**.  
It provides information about landmarks, cultural spots, and travel tips for visitors.  
The goal is to practice **web development skills**, layout design, and responsive styling.

---

## Features
- Overview of Berlin landmarks and attractions  
- Clear navigation menu for different sections  
- Responsive design for desktop and mobile devices  
- Organized sections for sightseeing, culture, and travel tips  

---

## Technologies Used
- HTML5  
- CSS3  
- CSS Grid and Flexbox for layout and responsiveness
